Target Checkout Lists

Create active Target checkout lists that users can choose from. Each list supports up to 29 SKUs in sku;name;price format. The SKU is matched to the website catalog name. If your price is blank, the website catalog price is used when available; otherwise it stays blank/no limit.

One item per line. Maximum 29 lines per checkout list. Product names are corrected from the website catalog by SKU when available. Missing SKUs are automatically added as catalog placeholders so you can finish the product name/credits.

Paste the generated website webhook URL into your bot. When a checkout comes in, the website finds the user, deducts credits, stores the order, sends a Discord webhook to the super admin, and also sends a second Discord webhook to the owner admin if that user belongs to one.
