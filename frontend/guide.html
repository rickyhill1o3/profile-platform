The Shore Shack TCG

User Guide

Everything a user needs to know before the next drop.

Home Login Sign Up

1. Access the website

Go to https://www.theshoreshacktcg.com. If a browser has old DNS or SSL cache, open the site in a private window or toggle Wi-Fi off and back on.

2. Create your account

Click Sign Up, enter your email, password, and invite code if one is required. You will receive a welcome email after signup.

3. Build your store profiles

Create profiles for the stores you want to run:

  • Amazon for Amazon-specific checkout details.
  • Target for Target-specific checkout details.
  • Walmart for Walmart-specific checkout details.
  • General for Supreme or any release that does not fit a store-specific profile.

Your saved profiles are what the software uses to shop. They contain the checkout information the platform needs when the release starts.

4. Select your products

Open the Products section and switch between the Amazon, Target, Walmart, and General tabs. Mark the products you want the system to run for, then save your preferences.

Important: your profiles tell the platform how to shop, and your selected products tell it what to shop.

5. Request SKUs if something is missing

If a product is not in the list yet, use the SKU request form. The software will try to look up the SKU, add it to the correct catalog, and pull in the best available title and price data automatically.

6. Watch live countdowns

Countdown cards show Walmart, Target, Supreme, and other general release windows. Admins can also post one-off release names for general drops.

7. Password reset

If you forget your password, use the Forgot Password page. A reset email will be sent to your inbox with a secure link.

Quick reminders

Profiles first

Create your store profiles before you expect the bot to run.

Selections matter

Only products you select in the Products page are eligible to run for your account.

Match store to profile

Use the Walmart profile for Walmart drops, the Target profile for Target drops, and so on.

General profiles

General profiles are used for Supreme and other non-store-specific general releases.
