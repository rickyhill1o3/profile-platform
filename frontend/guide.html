Complete User Guide

Follow these steps in order to prepare your account for releases.

Home Open Dashboard
Recommended order

1. Create profiles → 2. Assign stores → 3. Add store login and IMAP/2FA → 4. Turn stores on → 5. Select products → 6. Confirm green sync status.

1. Create your account

Sign up with your invite code, verify that you can log in, and open the User Dashboard. The guided setup card at the top checks your real account and shows what is incomplete.

2. Build your first profile

Open Profiles and click Create Profile. Enter the shipping address, billing address, phone number, email, and payment information that should be used at checkout.

You may create multiple profiles. Give each profile a clear name so you and the admin can identify it.

3. Assign stores to the profile

While creating or editing a profile, select every store that should use that profile. Store assignments determine where the profile appears and which store credentials are required.

4. Add retailer login credentials

For each assigned store, enter the retailer account email and password. Target, Walmart, and Sam’s Club generally also require an email app password for verification codes. Amazon generally requires an authenticator secret.

5. Set up IMAP / email app passwords

IMAP allows the platform to retrieve one-time verification codes sent by retailers. Use an app password, not your normal email password.

Gmail

  1. Turn on Google 2-Step Verification.
  2. Open Google Account → Security → App passwords.
  3. Create an app password named “The Shore Shack”.
  4. Paste the 16-character app password into the profile’s IMAP/App Password field.

Outlook / Hotmail

  1. Turn on two-step verification in Microsoft Account Security.
  2. Open Advanced security options and create an app password.
  3. Paste it into the IMAP/App Password field.

Yahoo

  1. Open Yahoo Account Security.
  2. Generate an app password for “Other App”.
  3. Paste it into the IMAP field.

iCloud

  1. Open Apple ID account settings.
  2. Create an app-specific password under Sign-In and Security.
  3. Paste it into the IMAP field.

6. Set up authenticator / 2FA

The platform needs the permanent setup secret behind the rotating six-digit code.

  1. Open the retailer’s security settings and choose authenticator app setup.
  2. When the QR code appears, choose “Can’t scan?”, “Enter setup key”, or “Show secret key”.
  3. Copy that permanent key into the website’s 2FA Secret field.
  4. Also scan the QR code into your authenticator app.
  5. Finish retailer verification and save the profile.

Do not paste the current six-digit code. It expires every 30 seconds.

7. Turn stores on

Open Store Run Status. Turn on every store you authorize The Shore Shack to run. A paused store will not run even if profiles and products are configured.

8. Select products

Open the store page and select the products you want targeted. For Target, selecting an admin checkout list adds every SKU in the list. Unselecting the list removes those products, so reselect individual products you still want afterward.

Amazon supports multiple products, quantities, and one primary item when those controls are available.

9. Understand red and green notices

Red: you changed profiles or product selections and the admin still needs to update the bot export.

Green: the admin has copied your latest information.

10. Final check before a release

  • At least one complete profile exists.
  • The profile is assigned to the correct store.
  • Retailer login credentials are complete.
  • IMAP or authenticator secret is present when required.
  • The store is active in Store Run Status.
  • Products are selected.
  • Profile and product notices are green.
  • Your credit balance is sufficient.

Common problems

My store is not running

Check Store Run Status, assigned stores, credentials, product selections, and credit balance.

Email verification codes fail

Generate a fresh app password, confirm the mailbox matches the retailer account, save the profile, and wait for the admin to update the profile export.

Amazon 2FA fails

Reconfigure authenticator and save the permanent setup key rather than the temporary six-digit code.

I changed something

A red notice is expected until the admin copies the update.
