Public onboarding guide

How to use The Shore Shack

This guide is viewable by anyone. Profile management, account data, and private actions still require login.

1. Access the website

Open The Shore Shack home page to learn how the platform works. From there, users can choose to sign up, log in, or read this guide before creating an account.

2. Create an account

  1. Open the Sign Up page.
  2. Enter your email, password, and invite code if one is required.
  3. Submit the form to create your account.
  4. After signing up, a welcome email is sent to confirm you are in the system.

Existing users can always use the Login page. If needed, the Forgot Password page can send a reset email.

3. Create profiles for each store

Profiles are the saved checkout details the platform uses when it runs for your selected products.

Amazon Profile

Create this for Amazon products you want the platform to shop for.

Target Profile

Create this for Target runs and keep your Target information up to date.

Walmart Profile

Create this for Walmart releases and Walmart product runs.

General Profile

Use this for General runs such as Supreme or other store types that use the general flow.

Always make sure your profile information is accurate. The system can only use the information you save.

4. Select products in the Products section

The Products section is where users choose exactly what items the platform should run for.

  1. Open the Products area after logging in.
  2. Choose the store tab you want to work in: Amazon, Target, Walmart, or General.
  3. Review the list of available items.
  4. Select the products you want the system to run for.
  5. Save or leave the selections active so the platform knows what to target.

Important: If a product is not selected, the platform should not run for it on your account.

5. How the platform uses your profiles

Your saved profiles are used to shop for the items you selected in the product selection page.

Create profileTell the platform how to check out.
Select productTell the platform what to run for.
Platform runsThe matching store profile is used for the selected items.

Example: if you create a Target profile and select three Target products, the platform uses your Target profile when running for those three Target products.
