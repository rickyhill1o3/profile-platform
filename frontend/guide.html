How to use The Shore Shack TCG

This guide is public. Anyone can view it before signing up. Private areas like profiles, product selections, and admin controls still require login.

1. Create an account

Go to Sign Up, enter your email and password, and complete account creation. New users receive a welcome email after signup.

2. Build your store profiles

Create the profiles you want the platform to use during checkout. Profiles store the checkout information that gets used when the system runs.

  • Amazon profile — used for Amazon releases.
  • Target profile — used for Target releases.
  • Walmart profile — used for Walmart releases.
  • General profile — used for Supreme, Pokémon Center, and other custom drops.

3. Select products

Go to the Products section and select the items you want the system to run for. Your saved profiles tell the platform how to buy. Your product selections tell it what to buy.

  • Amazon tab — select Amazon products.
  • Target tab — select Target products.
  • Walmart tab — select Walmart products.
  • Supreme tab — select Supreme items or the Supreme next release option.
  • Pokémon Center tab — select Pokémon Center items or the next release option.
  • General tab — select custom release items or a next release entry when exact SKUs are not available yet.

4. Request missing SKUs

If you do not see a Walmart, Target, or Amazon product in the product list, use the SKU request form. The system can attempt to look up the SKU, enrich the catalog, and add the product with a title and price.

5. Follow release countdowns

The release countdowns board shows upcoming Walmart, Target, Supreme, Pokémon Center, and custom general releases. Admins can also attach related products to each countdown so users know what the timer is for.

How your account is used during releases

When a release goes live, the system uses the profiles you saved to attempt checkout for the products you selected. If you selected “Shop Next Release” for a site, that tells the platform you want to run the next upcoming drop for that site even when the exact product list is not known yet.
