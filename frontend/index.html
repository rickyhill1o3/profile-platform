What the platform actually does

Users save store-specific profiles with checkout information, then select the products they want the software to run for. During releases, those saved profiles are used to attempt checkout for the selected products. If the exact drop list is not known yet, users can still mark Walmart, Target, Supreme, Pokémon Center, or General “next release” entries so they are queued for the upcoming drop.

Public onboarding

Anyone can view the homepage and user guide without logging in. Private areas like profiles, products, and admin tools remain protected behind login.

Admin controls

Admins can create countdowns, attach related products to them, enrich the catalog by SKU, and review user product selections.
