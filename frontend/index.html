How it works

Create profiles, select products, and the platform uses those saved profiles during runs for your chosen items.

What the platform does

Everything users need to get ready before a run.

Public visitors can learn how the platform works before creating an account. Profile details and private account actions stay behind login.

1

Create store-specific profiles

Users can set up dedicated profiles for Walmart, Target, Amazon, and General shopping flows with saved checkout details.

2

Select exact products

The Products page is where users decide what items the system should run for. No selection means nothing will run.

3

Use saved profiles during runs

The profiles users build are the profiles used to shop for the items they selected in the product selection page.

01

Make an account

Sign up, verify your login details, and access the dashboard.

02

Build your profiles

Add store-specific information for Amazon, Target, Walmart, or General.

03

Choose your products

Open the Products section and mark the items you want the platform to run for.

04

Be ready for the run

When the platform runs, it uses the saved profile tied to the store and the products you selected.
